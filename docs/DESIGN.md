# 知乎风格论坛系统 - 需求分析与设计

## 一、核心功能需求

### 1. 用户系统
- 用户注册/登录
- 个人主页
- 关注/粉丝
- 用户头像、简介

### 2. 问答系统
- 发布问题
- 回答问题
- 邀请回答
- 问题分类/标签

### 3. 互动系统
- 点赞/踩
- 评论（对回答）
- 收藏
- 分享

### 4. 内容展示
- 首页推荐流
- 热门问题
- 关注动态
- 搜索功能

### 5. 其他
- 举报/审核
- 草稿箱
- 消息通知

---

## 二、技术选型

### 前端方案
考虑到部署便捷性和网络限制，采用**纯前端 + 静态部署**：

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3 + Vite | 轻量快速 |
| UI | TailwindCSS | 现代美观 |
| 状态 | Pinia | 简洁状态管理 |
| 路由 | Vue Router | SPA 路由 |
| 数据 | LocalStorage + JSON | 本地模拟后端 |

### 部署方案
- **GitHub Pages**: 免费静态托管
- **GitHub Actions**: 自动构建部署

### 后端方案（未来扩展）
- 可接入 Supabase / Firebase 提供真实后端
- 或部署 Node.js API 到 Vercel/Cloudflare Workers
